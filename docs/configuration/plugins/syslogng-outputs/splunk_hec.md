# Sending messages over Splunk HEC
## Overview

{{< highlight yaml >}}
apiVersion: logging.banzaicloud.io/v1beta1
kind: SyslogNGOutput
metadata:
  name: splunkhec
spec:
  splunk_hec_event:
    url: "https://splunk-endpoint"
    token:
      valueFrom:
          secretKeyRef:
            name: splunk-hec
            key: token
{{</ highlight >}}

More information at https://axoflow.com/docs/axosyslog-core/chapter-destinations/syslog-ng-with-splunk/
